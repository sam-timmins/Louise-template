Problem

This figure shows a simply supported beam of length L subjected to a sin curve varying load, with maximum magnitude q0, applied along its full length.

You should design a userform which contains an image of the beam, a frame for the input values, a command button to trigger the calculations, and a frame to output the results. 

The output should include:

  • the maximum displacement
  • the displacement at a point x (provided by user)
  • the rotation of the support.

All required equations are provided below.

Equations

It can be assumed that the beam has a solid rectangular cross-section, therefore the second moment of area is given by:

`I = frac(bd^3)(12)`

The maximum deflection in the centre of the beam is given by:

`delta_(max) = frac(q_0 L^4)(pi^4 EI)`

The angle of rotation (slope) of the beam at the supports is given by:

`theta_A= theta_B = frac(q_0 L^3)(pi^3 EI)`

The deflection at any distance x from the left-hand support is given by:

`delta_x= frac(q_0 L^4)(pi^4 EI) sin frac(pi x)(L)`

Where:

  • b = width of the beam
  • d = depth of the beam
  • L = length of the beam
  • E = Young's Modulus, you can take a default value for concrete of 30 GPa (30 kN/mm2)
  • q0 = Maximum magnitude of sin curve varying load applied along the length of the beam.
